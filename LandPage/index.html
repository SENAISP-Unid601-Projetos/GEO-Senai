Projeto

AutoSenai

Nosso projeto consiste em desenvolver um totem para mapear e orientar os alunos novos e veteranos do Senai. Este totem será instalado em locais estratégicos dentro da unidade Senai. O objetivo é facilitar a localização dos alunos dentro da escola, proporcionando uma experiência mais fluida de navegação pelo ambiente.

Quem nós somos?

Guilherme Lopes - Designer, Davi Ponce - Front-end, Rhuan Dhiego - Banco de dados, Gabriel Bento - Front-end e Lucas Oliveira - Back-end

O custo estimado para a implementação é determinado pelo uso de software e pela construção do totem de madeira, utilizando tecnologias como React Native, Banco de Dados e Java. Estamos planejando desenvolver o totem utilizando essas tecnologias e uma estrutura feita de madeira. A previsão é concluir o projeto até maio.

1. Objetivo

Garantir que 90% dos estudantes acessem informações acadêmicas relevantes por meio do totem.

2. Objetivo

Reduzir o tempo médio gasto pelos usuários para localizar salas em pelo menos 30% após a implementação do totem.

3. Objetivo

Alcançar uma taxa de utilização do totem de autoatendimento de 75% entre os estudantes e visitantes do Senai.
